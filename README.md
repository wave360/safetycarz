# Safety Car Z - Site Web

Site web professionnel pour Safety Car Z, entreprise spécialisée dans la location de voitures haut de gamme.

## 🚀 Technologies utilisées

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Déploiement**: Vercel

## 🛠️ Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/ton-utilisateur/safety-car-z.git
   cd safety-car-z
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

4. Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## 📦 Déploiement

Le site est prêt à être déployé sur Vercel. Suivez les instructions de la documentation Vercel pour un déploiement en production.

## 📄 Pages disponibles

- **Accueil** (`/`) : Page d'accueil avec présentation et call-to-action
- **À propos** (`/about`) : Histoire et valeurs de l'entreprise
- **Services** (`/services`) : Liste complète des services proposés
- **Blog** (`/blog`) : Articles et conseils
- **Contact** (`/contact`) : Formulaire de contact et coordonnées

## 🎨 Design

- Palette de couleurs : Bleu foncé (#1a2332), rose (#f8b4cb), rouge (#ff6b6b)
- Typographie : Inter (Google Fonts)
- Animations fluides avec Framer Motion
- Design responsive (mobile, tablette, desktop)
